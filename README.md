# SLN Hackathon — Starter-Projekt

Eine fertige SvelteKit-App mit Datenbank, vorbereitet zum Bauen mit **Claude Code**.

**Du machst zum ersten Mal mit?** → Öffne **[HACKATHON.md](./HACKATHON.md)** und folge
der Schritt-für-Schritt-Anleitung.

**Du bist aus dem Dev-Team?** Stack: SvelteKit (Svelte 5) + Drizzle ORM + SQLite (libSQL,
Datei `local.db`). Setup: `npm install && npm run db:push && npm run dev`. Architektur-
Leitplanken für Claude stehen in [CLAUDE.md](./CLAUDE.md). Ideen-Liste in
[tasks/IDEEN.md](./tasks/IDEEN.md).
